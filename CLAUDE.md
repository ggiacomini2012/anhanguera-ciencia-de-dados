# Git: Ciência de Dados

Repositório: `/metas/faculdade/ciencia-de-dados`

## Commits
- Um commit por aula — nunca agrupe aulas diferentes no mesmo commit
- Formato obrigatório: `<tipo>: S<N>-D<N>-U<N>-A<N>: <Título Curto>`
- Exemplo: `feat: S2-D1-U4-A2: Teste de Hipóteses`

## Push
- Um push por dia = um único commit (o mais antigo da fila)
- Demais commits ficam para os próximos dias (manter streak diário)
- Lembrar o Gui de fazer o push ao final do dia se houver pendências

## Auditoria
- Nunca invente nomes de disciplinas ou diretórios
- Pasta `-RENOMEAR`? Peça confirmação com base no portal da faculdade
- Antes do push: confirmar que `.env` está no `.gitignore`
