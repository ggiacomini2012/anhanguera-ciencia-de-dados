- Aprovação por disciplina: **60 pontos**.
- Composição típica informada (aproximada):
- Visualização de conteúdo: **10%**
- Exercícios: **15%**
- Trabalhos: **25%**
- Prova: **50%**

Observação: os pesos podem variar entre disciplinas, mas esse é o modelo-base para planejamento.

### Exceção: disciplinas de projeto

- A avaliação é centrada na **entrega de projeto** (ex.: arquivo PDF).
- O projeto recebe uma nota final e define aprovação/reprovação conforme as regras da disciplina.

### Implicação prática para priorização

- Prova continua tendo peso alto, mas **não carrega sozinha aprovação automática**.
- Para reduzir risco de reprovação, o ideal é acumular pontos antes da prova via visualização, exercícios e trabalhos.
